# header/ - Page Header Components

Dynamic header content based on current page context. **10 files**.

## Files

- `index.tsx` - Header dispatcher (selects component by page type)
- `banner.tsx` - Info/warning banners
- `auth/index.tsx` - Auth page header

### Main Headers (`main/`)
- `object.tsx` - Object page header with title and actions
- `empty.tsx` - Empty/minimal header
- `graph.tsx` - Graph page header
- `history.tsx` - Version history header
- `settings.tsx` - Settings page header
- `navigation.tsx` - Navigation page header
- `chat.tsx` - Chat page header

# selection/ - Block Selection System

Multi-block selection with keyboard and mouse support.

## Files

- `provider.tsx` - Selection provider with drag-select rectangle
- `target.tsx` - Selectable block wrapper

## Features

- Drag-to-select with threshold detection
- Keyboard modifiers: Shift (range), Ctrl/Cmd (toggle)
- Context menu handler registration
- Selection state map with parent node caching
- Integrates with focus management system
